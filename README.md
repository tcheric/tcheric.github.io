## Lists
### Initiaialising a List
```python
# Empty list
new_list = []
# Populated list
new_list = [1, 2, 3]
# Fixed length list
new_list = [-1] * list_length
```
### List Indexing and Slicing
```python
# First item
first_item = my_list[0]
# Nested list value
nested_item = my_list[0][1]
# Negative indexing
last_item = my_list[-1]

# my_list = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
# Index 2 to index 4
sliced_list = my_list[2:5] # ['c', 'd', 'e']
# Index 5 to end
sliced_list = my_list[5:] # ['f', 'g']
```
